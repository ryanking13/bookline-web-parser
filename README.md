# Naver Book Crawler
  - 네이버 책 DB를 사용하여 책의 대사들을 크롤링 하는 프로그램
  - 검색할 년도-월-주를 지정하면 해당 주 소설 영역 베스트셀러를 기준으로 검색함
  - 현재 교보문고 베스트셀러를 기준으로 검색함


    $ python web-parser.py

    검색할 년 월 주를 입력하세요(xxxx-xx-x) : 2017-10-1
